tleeaf UI

Pixel ACG Studio で使用している独自のデザイン言語、tleeaf UIの紹介です。


About

　Pixel ACG Studio では使用しているCSSなどのスタイル・デザインを統一させる目的で、独自のデザイン言語を決めました。そのデザインの仕様を紹介していきます。

Font

　フォントは基本的には本文はLINE Seed JP Rg、ページ内タイトルはNew Super Mario Font Uを使用します。
ページのCSS上ではLINE Seed JP RgはLINE_Rg、New Super Mario Font UはMario_Uと指定します。

Font Size

　フォントの大きさは基本的にH1などでも主張し過ぎないように設定されています。

H1: 32px


H2: 24px


H3: 18px


本文: 16px
